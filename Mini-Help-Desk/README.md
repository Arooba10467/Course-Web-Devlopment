This is a Mini Help Desk Project to help user add Ticket to submit their query and questions in Web Development, using HTML, CSS and JS
Objectives:
- View a list of tickets (with subject, description, and priority)
- Add a new ticket using a form
- Delete existing tickets
- Input validation for required fields
- Responsive layout for desktop and mobile screens

Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Flexbox for layout and styling)
- JavaScript (DOM manipulation and event handling)
